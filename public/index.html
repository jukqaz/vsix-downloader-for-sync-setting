사용 방법

  1. 터미널에서 node index.js check <extensions.yml 파일 경로> 명령을 실행하여 확장 프로그램 목록을 확인합니다.
  2. Open VSX에서 사용할 수 없는 확장 프로그램 목록이 표시됩니다.
  3. 웹 인터페이스에서 ID 또는 UUID로 확장 프로그램을 검색하고 다운로드할 수 있습니다.
  4. 다운로드한 VSIX 파일을 Open VSX 기반 IDE에 수동으로 설치합니다.

명령어 옵션

  • node index.js check <extensions.yml 파일 경로>: 확장 프로그램 목록을 확인합니다.
  • node index.js download <확장 프로그램 ID>: ID로 특정 확장 프로그램을 다운로드합니다.
  • node index.js download-by-uuid <UUID> -f <extensions.yml 파일 경로>: UUID로 특정 확장 프로그램을 다운로드합니다.
